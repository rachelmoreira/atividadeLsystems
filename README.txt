# atividadeLsystems
No arquivo,devem ser definidos,nessa ordem:

- O ângulo que será utilizado nos comandos para mudar a direção (esquerda ou direita)
- O axioma (a letra inicial,como começa),dado no exemplo como 'x'
- Regra 1
- Regra 2

Comandos da(s) regra(s):
- F -> desenhar o traço (drawForward)
- '-' -> mudança de direção para esquerda
- '+' -> mudança de direção para direita
- '[' -> push,salva posição atual
- ']' -> pop,iniciar nova sequência
